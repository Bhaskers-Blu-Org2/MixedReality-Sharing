    Interface IDiscoverySubscription

    Handle to an ongoing discovery subscription.

    Namespace: Microsoft.MixedReality.Sharing.Matchmaking
    Assembly: Microsoft.MixedReality.Sharing.Matchmaking.dll
    Syntax
    public interface IDiscoverySubscription : IDisposable
    Remarks

    A subscription is bound to the originating IDiscoveryAgent, but its methods/properties (including Dispose()) must be safe to use independently from the state of the agent - even from a different thread from the one where the agent is used.

    After the agent is disposed, the result of Resources eventually becomes empty, and Updated stops being raised.
